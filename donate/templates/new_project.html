Create A New Project

A project is something to which someone can donate. A project can have a defined lifetime with a set amount to be raised, or it can represent an area like Sewing or Electronics which has incoming donations and outgoing expendatures.

Start a new project by entering a name and a description. If this project is trying to raise a specific amount, enter that as the goal, and select the currency as well.
